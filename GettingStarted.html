Getting to know your Engduino

  • Microcontroller
  • 16 RGB LED lights
  • Sensors
  • Temperature sensor
  • Light sensor
  • Magnetometer
  • Infrared transceiver
  • User programmable button
  • Rechargeable 3.7V protected LiPo battery
  • Micro-SD card reader
  • Expansion connector
  • USB connector
  • On/off switch


The Arduino-Engduino IDE (Integrated development environment)

The IDE is used to write the code for the Engduino. We write our code in this app, verify the code to make sure that’s the syntax is correct and the we upload the code into the Engduino.

Installing the IDE (Integrated development environment) – Where you write your code!

  1. Go to http://www.engduino.org/html/Download.html
  2. Select the appropriate software depending on your OS
  3. Once downloaded, install the application.
  4. Once installed open the app (called Arduino-Engduino).


  5. You should now see this
  6. Now click on Tools -> Board -> EngduinoV3




  7. If your Engduino is connected the click on Tools -> Port -> “The available port”. This is to make sure that when we are uploading the code into the Engduino, the computer knows where to find it.
