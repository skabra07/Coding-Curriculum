C Cheat Sheet

Variables

If you want to store a value, you put it in a variable. To use a variable, first you have to declare it in the format type 

DataType variableName;


For example:
int myNumber;


At this stage, you can also initilise a variable, which means assign a value to the variable.

For example:
char myString = ‘cat’;


If you have mutliple variables of the same type that you are declaring but not initialsing, you can list them next to each other

For example:
float float1, float2, float3;
float1 = 3.235; float2 = 4.567; float3 = 3.0;

Input and Output

The two main ways you will input and output data is by asking the user to enter values and by displaying things on the console. You will need to put 

 #include "stdio.h"
at the top of your program to be able to use these functions.

To display something on the console, use 
printf( );


If you just want to print out a string, you can do 
printf(“Hello”);


If you want to print a variable, use this format 
printf(“%formattingCharacter”, variable);


For example:
int myNumber = 10;
printf(“My number is %d”, myNumber);

This would print out “My number is 10”

To make what you’re printing print on a new line, put 
\n
inside the speech marks

To take input from the user, use 
scanf(“%formattingCharacter”, &variableName”);


It’s a good idea to print something before using 
scanf
to let your user know to enter something

For example:
int favouriteNumber;
printf(“Please enter your favourite number: “);
scanf(“%d”, &favouriteNumber);

Arrays

Declaring an array in C is very similar to declaring a variable – the format for declaring an array is 

type arrayName[arraySize];


For example:
int myNumbers[5];


You can fill the array later on in different ways; either by filling all the spaces at once, or filling each index (position in the array) one by one

For example: 
myNumbers = {1, 2, 3, 4, 5};
Note the use of curly brackets
myNumbers[0] = 45;


Alternatively, when you declare the array, you can initliase it at the same time, remembering to use the curly brackets instead of the square brackets

For example: 
char myWords[ ] = {‘cat’, ‘dog’, ‘mouse’};


You can access the array elements by index, and do things like assign the value in the index to a variable or print the value in the index to the console

For example:
char myFavouriteAnimal = myWords[1];


You can make multi-dimensional arrays by declaring them like this 
type arrayName[size1][size2][size3]…
where 
size1
is the size of the first dimension, 
size2
is the size of the second dimension etc.

A two-dimensional array with 3 rows and 2 columns might look like this for example;
int a[2][3];

If/Else Statements

An If/Else statement will do 1 thing if a given condition is true, or do something else if that statement is false. They are constructed like this: 

if(statement to be true or false){
 do one thing;
}
else if(another statement to be true or false){
  do another thing;
}
else{
  do a different thing;
}


You can have as many else if clauses as you want, but only one else clause in any if statement. You could have 0 else if statements and/or 0 else statements too if you need.

The statements to be true or false often compare values, using operators: 

AND &&
EQUALS ==
NOT EQUAL !=
OR ||


There is no between operator in C.

In C, you can’t compare strings using 
==
, so you have to use 
strcmp(str1, str2) or strncmp(str1, str2, n)
functions instead. These are library functions, which means they are written by somebody else for you to use. At the top of your program you must put 
#include "stdlib.h"
to access any library functions. 

strcmp
and 
strncmp
both take in two strings to compare to each other, but 
strncmp
also takes in a number n, which is the number of characters from each string you want to compare. They both return either 0 if the strings are equal, or 1 if the strings are not equal

For example: 
char myWord = ‘cat’;
char anotherWord = ‘catdog’;

int isThisEqual = strcmp(myWord, anotherWord);
int isThatEqual = strncmp(myWord, anotherWord, 3);


In this example, isThisEqual would be 1, and isThatEqual would be 0

Loops

There are two main types of loop used in C, a for loop and a while loop. A for loop will do a specific task a certain number of times, whereas a while loop will do a specific task indefinitely if a certain condition is true.

A for loop is made up of a loop variable, a condition for the loop variable, and an increment or decrement statement for the loop variable.

For example: 

for(int i = 0; i 10; i = i +1){
  count = count + 100;
}


This piece of code will add 100 to a count variable 10 times. The loop variable i starts at 0. Each time the code in the for loop is run, i increases by 1. This will continue happening while i is less than 10; i.e., when i reaches 10, the code inside the for loop won’t run and the loop will have finished.

Instead of writing i = i + 1, you can write i++. Similarly, you can write i-- to make i decrease by 1 each time the code runs. You don’t have to make i increase or decrease by 1 each time, you can pick any number you want e.g. i = i + 20

A while loop is simpler, and is only made up of a condition.

For example: 
while(age 19){
  printf(“I am still a teenager”);
  age = age + 1;
}


This code will carry on printing “I am still a teenager” as long as age is less than 19. Once age reaches 19, the while loop will finish.
