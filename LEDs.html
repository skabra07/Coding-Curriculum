LEDs

Suyash the lion and his friends want to see who can run further in ten seconds but there is no way they can measure the time. Good thing Suyash has an Engduino and found this grrrrreat website. We will show you in a simple tutorial how to turn your Engduino into a timer.

#include "EngduinoLEDs.h"
void setup() {

}
void loop() {

}

In C, there exist libraries with predefined functions. This helps us from writing those functions again. To incorporate these libraries we use the key word #include "file_Name.h". For the engduino, the super awesome programmers at UCL already made libraries so that lions ,like Suyash, can use it without making their own functions. Mostly, these functions help to control the sensors of the engduino. In this case, we included a library for using the LEDs on the engduino.

#include "EngduinoLEDs.h"
void setup() {
  EngduinoLEDs.begin();
  EngduinoLEDs.setAll(GREEN);
}

void loop(){
}

To call a function of the library we use the syntax libraryName.functionName().
EngduinoLEDs.begin() - This function opens the LEDs so that they can receive voltage where a voltage of HIGH is on and LOW is off.
EngduinoLEDs.setAll(GREEN) - This function turns all the LEDs green.
You can upload your program right now and you should have a cool green light. But green is not a creative color so let us quickly move on. So in the beginning all the LEDs will be on and then we will turn off one LED after another, until all are off. We need the function "set.LED(LED_NUMBER, MODE)" to control a single light, whereas LED_NUMBER is a value from 0 to 15. The amount of time between turning off one led and another is called the delay. this delay is crucial for us to build our timer . Delay takes in a time in milliseconds, so we will have to calculate our delay by taking into account the total time (in ms) divided by 16. I.n this case the total amount of time is 10ms.

Voilà ! we have a timer. 

#include "EngduinoLEDs.h"
void setup() {
  EngduinoLEDs.begin();
  EngduinoLEDs.setAll(GREEN);
}
void loop(){
  for(int i=15 ; i>=0 ; i--){
    EngduinoLEDs.setLED(i,OFF);
    delay(625);
  }
}

The timer is pretty much done, but wouldn't it be handy to have some sort of alarm going off, once the timer has finished? Well, the Engduino does not have sound but we can create a flashing light. Grrrreat. We can use a loop that will run forever onces the timer is done. In the body we will have a simple stating turning the LEDs on and off in a short amount of time.

#include "EngduinoLEDs.h"
void setup() {
  EngduinoLEDs.begin();
  EngduinoLEDs.setAll(GREEN);
}
void loop(){
  for(int i=15 ; i>=0 ; i--){
    EngduinoLEDs.setLED(i,OFF);
    delay(625);
  }
  while(1) {
    EngduinoLEDs.setAll(RED);
    delay(10);
    EngduinoLEDs.setAll(OFF);
    delay(10);
  }
}

And there we go, now the race can finally begin! The friends realize that they want to be able to quickly change the amount of time the timer takes in. Summary:

  • Always include the EngduinoLEDs.h library or you won’t be able to use the LEDs in the program
  • Make sure to turn on the LEDs that you will use before writing your code
  • EngduinoLEDs.setAll(color) = sets all the leds to that color
  • EngduinoLEDs.set(i,color) where i is the pin number of LED.
  • The full set of colours available to you is RED, GREEN, BLUE, YELLOW, MAGENTA, CYAN, WHITE and, although it's not strictly a colour, OFF.
  • To know the pin of a LED there is a small number written next to the LED.
  • You can also control the brightness by selecting a number that ranges from 0 (= off) to 15 (=brightest). To set the LEDs to a fairly dim YELLOW, we use: EngduinoLEDs.setAll(YELLOW, 2);
