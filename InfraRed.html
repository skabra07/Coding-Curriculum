Infra Red Sensor

Suyash the lion is moving into his new cave which has Engduinos all over the walls. He wants to change the colour of the LEDS in each Endguino to make a pretty wall display. Luckily he has an extra Engduino that can be used as a remote to control the lights of the other Engduinos on the walls. In this tutorial we will show you how to create a light remote that can change the lights of another Engduino! Suyash’s remote can turn the other Engduino’s LEDs into one of the 4 colours : RED, BLUE, YELLOW or GREEN. In fact, by holding the button and releasing it at the desired colour, the remote will send the selected colour to the other Engduino, and that will have all its LEDS changed to that colour. The Engduino comes with a built in IR sensor. A communication between two Engduinos is possible due to the IR sensors which can transmit or receive information. The remote Engduino will be the sender and each of the Engduinos on the wall will be the receiver. The library files to include the IR Sensor and the LEDs function that we will use:

#include "EngduinoLEDs.h"

Let’s start by writing a code to turn the engduino into a remote. First we need to turn on the IR sensor by calling the begin function so our engduino can start sending and receiving information. We also need to turn on the button so we can use it to select a colour and send it. Then, we should add these variables outside of the functions, right after the include statements 

char message;
int i;
char lights[4] = {'r', 'g', 'b', 'y'};

The message variable will hold a single character for a colour. int i is the itereator that will be used in the loop. The lights array stores the different characters that stand for each colour Our code will turn into this: 

#include "EngduinoLEDs.h"
#include "EngduinoButton.h"
#include "EngduinoIR.h"

char message;
int i;
char lights[4] = {'r', 'g', 'b', 'y'};

void setup() {
	EngduinoLEDs.begin();
	EngduinoButton.begin();
	EngduinoIR.begin();
}

Next we need to create an if/else statement in the main loop that will specify whether the button is pressed or not. If the button is pressed the loop will iterate consecutively through our lights until the button is released. When the button is released the selected colour will be the message that is transmitted. While the button is not pressed, the message will be continuously sent via the IR sensor. 

void loop() {
	if(EngduinoButton.isPressed()){
		for(i=0; i 4; i++){
			message = lights[i];
			setLights(message);
			delay(500);
		}

		if(EngduinoButton.isPressed() break;

	}

	else{
	EngduinoIR.send(message);	//send message
	}
}

void setLights(char light){
	switch(light){
		case 'r':
		EngduinoLEDs.setAll(RED);
		break;
		case 'g':
		EngduinoLEDs.setAll(GREEN);
		break;
		case 'b':
		EngduinoLEDs.setAll(BLUE);
		break;
		case 'y':
		EngduinoLEDs.setAll(YELLOW);
		break;
		default:
		EngduinoLEDs.setAll(OFF);
		break;
		}
}

The set lights function is a helper function that takes in one of the characters and displays the corresponding colour.engduinoIR.send(MESSAGE) is the function that will send the IR message.
