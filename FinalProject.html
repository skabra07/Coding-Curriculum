The Final problem

Let's build a game!

Now that you have learned how to use the Engduino with the help of Suyash the Lion and his friends, we want you to create something great. Create a version of the memory game 'simon' using various sensors of the Engduino and never be bored again. The rules are quite simple. You have 4 different fields with the colours red, green, blue and yellow. First a random field will light up. The player then has to light that field up. After that another random light is added to the order. The player wins when he has successfully reproduced an order of 10 (or more if you want to make it harder).

Hints

  • Use the accelerometer for controls.
  • Don't forget to choose a seed when using random().


More info:

  • https://en.wikipedia.org/wiki/Simon_(game)
  • https://www.arduino.cc/en/reference/random

Solution

Before checking out the solution please try and attempt the problem youself. If you are stuck, try using your friends help. After all Suyash the Lion was able to solve all the problems using his friends help.
The solution is file called final_project_solution.ino. After downloading the file open the Arduino-Engduio IDE (were you write your code). Then go to File->Open. Select the file you downloaded.
